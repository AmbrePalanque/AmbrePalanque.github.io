Remembex was born during a back-end development project. I developed this application in Ruby on Rails. It allows to list its "exs" according to their defects or their qualities. Would you be ready to start the adventure again with one of them?

This application is a reminder of the romantic relationships of the past. It is possible to create a profile, and to enter various elements such as a description, negative and positive points, a photo or illustration, a description and especially three notes. The average of these three marks will allow us to sort the pro- grams in a decreasing way. You can edit or delete a profile. Since the application is intended to be private, it is not possible to share the information. Each user can create an account and only have to enter the profiles. This project allowed me to learn how to code in Ruby on Rails, and although this language greatly facilitates the back end of development, it allows a great understanding of the other languages on the server side thanks to the MVC architecture which allows Structure all the elements.
