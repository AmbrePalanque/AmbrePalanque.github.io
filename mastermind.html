During the module mobile development in SWIFT I had the opportunity to create my first application by creating a Mastermind. We have formed a group of three students to carry out this project within four days.

For the game to be functional, one must start by generating a secret randomly. There are several ways to proceed. We chose to do two functions: a first function to generate random numbers, and a second one that will store these numbers to create a combination of 4 elements.
